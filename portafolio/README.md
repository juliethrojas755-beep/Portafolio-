## 🚀 Tecnologías Utilizadas

Para garantizar un rendimiento excelente, carga ultra rápida y control absoluto sobre los estilos y comportamientos, este portafolio fue desarrollado utilizando únicamente tecnologías nativas (Vanilla):

*   **HTML5 Semántico**: Para una estructura accesible (A11y), óptimo posicionamiento orgánico (SEO) y compatibilidad multi-dispositivo.
*   **CSS3 Avanzado (Vanilla)**: Con sistema de variables de diseño, layouts flexibles (Flexbox y CSS Grid), glassmorphism, degradados elegantes y animaciones clave (`@keyframes`).
*   **JavaScript Vanilla (ES6)**: Para micro-interacciones dinámicas, efectos de escritura, observadores de scroll (Intersection Observer) para animaciones fluidas y contador numérico interactivo.

> [!NOTE]
> Este proyecto **no utiliza frameworks** de estilos como Bootstrap o Tailwind, ni librerías adicionales de Javascript. Todo el código es propietario y está perfectamente estructurado y comentado para facilitar su mantenimiento.

## 🎨 Características de Diseño y UX/UI

1.  **Navbar Fija & Scroll Spy**: Barra de navegación con efecto Glassmorphic (`backdrop-filter: blur`) que cambia sutilmente de tonalidad al hacer scroll y resalta de forma automática la sección visible actual.
2.  **Hero Impactante**: Presentación profesional que incluye un texto dinámico animado (Typewriter) y mi fotografía perfil enmarcada en un borde con degradado azul/morado, sombra elegante, brillo glow sutil y animación flotante suave.
3.  **Sobre mí & Estadísticas Animadas**: Explicación de mi perfil profesional acompañada de tarjetas con contadores de métricas que incrementan dinámicamente al entrar en pantalla.
4.  **Línea del Tiempo (Timeline)**: Cronología moderna que conecta mi formación académica desde el bachillerato técnico hasta mi especialización técnica actual.
5.  **Tarjetas de Habilidades**: Tarjetas con glassmorphism ligero, efecto flotante, sombras elegantes y transiciones al pasar el mouse, utilizando logotipos SVG oficiales de cada tecnología.
6.  **Proyectos Detallados**: Grid responsivo con tarjetas que detallan la descripción del proyecto, las tecnologías clave, los aprendizajes adquiridos o aportes de diseño de experiencia de usuario, y enlaces directos a código y demo.
7.  **Cursor Interactivo**: Un cursor personalizado y dinámico para pantallas de escritorio que se expande e interactúa al pasar por encima de elementos interactivos (enlaces y botones).

## ⚙️ Configuración y Uso Local

Para abrir y visualizar este proyecto localmente en tu ordenador, sigue estos pasos:

1.  **Clona este repositorio**:
    ```bash
    git clone https://github.com/juliethrojas755-beep/BORRADOR-PROYECTO-JAVA.git
    ```
2.  **Accede a la carpeta**:
    ```bash
    cd BORRADOR-PROYECTO-JAVA/portfolio
    ```
3.  **Añade tus imágenes**:
    *   Copia tu fotografía profesional en `assets/images/` con el nombre `profile.jpg`.
    *   Copia las capturas de tus proyectos en `assets/images/` como `project1.png`, `project2.png` y `project3.png`.
    *   *Nota:* Si no añades las imágenes inmediatamente, el sitio cargará las hermosas ilustraciones vectoriales SVG que he diseñado por defecto para que la web luzca terminada.
4.  **Abre el proyecto**:
    *   Haz doble clic sobre el archivo `index.html` para abrirlo en cualquier navegador web, o utiliza la extensión **Live Server** de Visual Studio Code para un refresco en tiempo real durante tus pruebas.

## 🌐 Despliegue en GitHub Pages

Este proyecto está completamente optimizado para ser alojado de forma gratuita en **GitHub Pages**:

1.  Crea un nuevo repositorio público en GitHub (por ejemplo, `portfolio`).
2.  Sube los archivos de la carpeta `portfolio/` a la rama principal (`main` o `master`).
3.  En la configuración de tu repositorio en GitHub (*Settings*):
    *   Ve a la sección **Pages** (en la barra lateral izquierda).
    *   En **Build and deployment**, bajo **Source**, selecciona `Deploy from a branch`.
    *   Bajo **Branch**, selecciona tu rama principal (ej. `main`) y la carpeta raíz `/root`.
    *   Haz clic en **Save**.
4.  ¡Listo! En unos minutos, tu portafolio estará público en la URL asignada: `https://<tu-usuario>.github.io/<nombre-repositorio>/`
